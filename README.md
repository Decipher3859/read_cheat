# Bionic Reading Browser Extension

!> [!CAUTION]

> Work in Progress
> This project is in an early stage. Expect breaking changes, missing features and incomplete documentation

Minimal scaffold for a browser extension based on Manifest V3.

## Purpose

This is a side project that accompanies us at the beginning of our retraining as IT specialists. Since we have different levels of prior knowledge, the expectations vary. Our goal is not only to learn programming, but also to work effectively as a team and to manage projects in a structured way through to completion.

We started with this idea when we were introduced to bionic reading. In the long term, the project is intended to provide a variety of tools to make reading websites easier.

## Features (Conceptual)

- Website implementation of Bionic Reading patterns
- Focus mode to avoid distraction from currently read paragraph
- Extension menu for mode toggles and options
